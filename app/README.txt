AUDIO STREAMING SERVICE
Alex Keeler, Amie Nguyen, Lorenzo Ramirez
CS-351 Cloud Computing Final Project

SETUP INSTRUCTIONS:
Install required modules through pip
Setup flask application using this guide:
https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world
Instead of using microblog.py, use app.py

Setup a MySQL database using audiostream.sq
Create a user 'audiouser' with password 'audiouser' that is given privilege to the audiostream database

Access application through the browser with the URL localhost:5000

Required modules:
flask
flask-bootstrap
flask-wtf
mysql-connector-python
boto3
pywave
mutagen
